# ㈜에스에이치푸드 통합 ERP

온라인 유통 전문 이커머스 회사 ㈜에스에이치푸드의 통합 ERP 시스템입니다.

## 기능

- 📊 대시보드 — 매출·미수금·재고 현황
- 📋 매출전표 — 거래처별 특별단가 자동 적용
- 💰 정산·미수금 관리
- 📦 재고 관리 (3PL 연동 예정)
- 📈 손익계산서 · 재무상태표
- 🧾 부가세 신고자료
- 📅 월마감·결산
- 📡 채널별 분석 (스마트스토어/쿠팡/홈쇼핑/B2B)
- 🤖 AI 경영 질의
- 👥 직원 계정 관리 (권한별 접근 제어)

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포

GitHub에 push하면 GitHub Actions가 자동으로 빌드 & 배포합니다.

## 기술 스택

- React 18
- Vite 5
- Supabase (DB 연동 시)
